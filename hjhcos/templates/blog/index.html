块级元素

块级元素独占一行，高度、宽度可以控制且不会影响同级块元素。

display: block;

 

行内元素

可以多个标签存在一行，对宽高属性值不生效，完全靠内容撑开宽高。margin、padding、border 作用于该元素但父级不受影响、会影响同级行内元素或父级同级下的行内元素。 

xxxxxxxxxx
display: inline;

 

行内块元素

和相邻行内块在一行上，宽度为自身内容宽度。

xxxxxxxxxx
display: inline-block;

 

嵌套规则

  • 块元素可以包含行内元素或某些块元素
  • 行内元素不能包含块元素，只能包含其它的行内元素
  • h1~h6、p、dt 只能包含行内元素，不能再包含块级元素
  • 块级元素与块级元素并列，行级元素和行级元素并列

contenteditable

元素内容是否可编辑。

xxxxxxxxxx
<p contenteditable="true">这是一个可编辑的段落。</p>

 

data-*

存储页面或应用程序的私有自定义数据。

xxxxxxxxxx
<script>
function showDetails(animal) {
    var animalType = animal.getAttribute("data-animal-type");
    alert(animal.innerHTML + "是一种" + animalType + "。");
}
</script>
<ul>
  <li onclick="showDetails(this)" id="owl" data-animal-type="鸟类">喜鹊</li>
  <li onclick="showDetails(this)" id="salmon" data-animal-type="鱼类">金枪鱼</li>  
  <li onclick="showDetails(this)" id="tarantula" data-animal-type="蜘蛛">蝇虎</li>  
</ul>

form 事件

由 HTML 表单内的动作触发的事件（应用到几乎所有 HTML 元素，但最常用在 form 元素中）。

onblur

元素失去焦点时运行的脚本。

xxxxxxxxxx
<script>
function upperCase()
{
var x=document.getElementById("fname").value
document.getElementById("fname").value=x.toUpperCase()
}
</script>
<label>
    请输入您的姓名（英文字符 失去焦点小写自动转大写）：<input type="text" name="fname" id="fname" onblur="upperCase()">
</label>
onchange

在元素值被改变时运行的脚本。

xxxxxxxxxx
<script>
function checkField(val)
{
alert("输入值已更改。新值是：" + val);
}
</script>
<input type="text" name="txt" value="Hello" onchange="checkField(this.value)">

按键事件

由键盘或类似用户动作触发的事件。

onkeydown

在用户按下按键时触发。

xxxxxxxxxx
<script>
function displayResult() {
    var x;
    if(window.event)
        {
        x=event.keyCode;
        }
    else if(event.which)
        {
        x=event.which;
        }
    var keychar=String.fromCharCode(x);
    alert("按键 " + keychar + " 被按下");
}
</script>
<input type="text" onkeydown="displayResult()">
onkeypress

在用户敲击按钮时触发。

xxxxxxxxxx
<script>
function displayResult()
{
var x;
if(window.event) // IE8 以及更早版本
    {
    x=event.keyCode;
    }
else if(event.which) // IE9/Firefox/Chrome/Opera/Safari
    {
    x=event.which;
    }
keychar=String.fromCharCode(x);
alert("按键 " + keychar + " 被按下");
}
</script>
<input type="text" onkeypress="displayResult()">
onkeyup

当用户释放按键时触发。

xxxxxxxxxx
<script>
function displayResult()
{
var x=document.getElementById("fname");
x.value=x.value.toUpperCase();
}
</script>
<input type="text" id="fname" onkeyup="displayResult()">
